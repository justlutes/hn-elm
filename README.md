# HN-ELM

Hackernews clone built with Elm, Typescript, and Web components

## Project Structure

```sh
src/
├── assets # Static assets
├── css # Sass files
├── elm # Elm project structure
├── js # Typescript project structure
```

## Available commands

### Developing

```shell
  yarn start
```

### Building

-- TSLint

```shell
  yarn build
```

### Elm-Typescript-Interop

```shell
  yarn generate
```
